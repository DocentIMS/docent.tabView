docent.tabView
==============================================================================

A tabbed view for dexterity content types

Features
--------

- Converts fieldsets into a tabbed view

Installation
------------

Install docent.tabView by adding it to your buildout::

    [buildout]

    ...

    eggs =
        docent.tabView


and then running ``bin/buildout``
